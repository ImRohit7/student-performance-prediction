# Student Performance Prediction

## Problem Statement
Student performance is influenced by multiple academic and personal factors. 
This project aims to build a Machine Learning model that predicts whether a student will pass or fail based on their academic data.

## Objective
- Analyze student academic data
- Build a predictive model using Machine Learning
- Evaluate model performance

## Technologies Used
- Python
- Pandas
- Scikit-learn
- Matplotlib
